\## 🇬🇧 English



\# 🐍 SnakeGame



Classic Snake built with Python and pygame. Nothing fancy — just the game working well, with a start menu, adjustable speed, border toggle, pause screen, touch support and a live timer.



---



\## Features



\- 10 speed levels so you can go from chill to chaotic

\- Border mode (hit the wall and you're dead) or borderless (wrap-around)

\- Pause menu with a semi-transparent overlay

\- Works with keyboard, mouse and touchscreen (swipe gestures)

\- Info bar at the top showing your score and time

\- Menu is fully keyboard-navigable too



\## Requirements



\- Python 3.8+

\- pygame



\## Setup



```bash

git clone https://github.com/jaimefg1888/snake.git

cd snake

pip install -r requirements.txt

```



\## Run it



```bash

python snake.py

```



\### Controls



| Action | Keyboard | Touch |

|---|---|---|

| Move | ↑ ↓ ← → | Swipe |

| Pause | Space | ⏸ button |

| Confirm | Enter | Click |

| Quit | Escape | — |
